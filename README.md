# dis-bot
Discord bot connect with chatgpt3 to give reasponse and chat with you. use dalle-2 to generate image from text.

## Install

```bash 
git clone
cd dis-bot
pip install -r requirements.txt
```

## Usage

```bash
python main.py
```
